# API de Autenticação

Repositório criado para armazenar o código da API de autenticação com banco de dados. Esta API é responsável por gerenciar o processo de autenticação dos usuários, incluindo registro, login e gerenciamento de sessões.

## Funcionalidades
- Registro de usuários
- Login de usuários
- Logout de usuários
- Proteção de rotas com autenticação

## Tecnologias Utilizadas
- Python
- Flask
- SQLAlchemy
- JWT (JSON Web Tokens)
- MySQL
- Docker (para containerização do banco de dados)
- bcrypt (para hashing de senhas)
